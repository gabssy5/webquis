Dokumentasi Rancang Bangun Game

Bagian ini memaparkan rancangan teknis pembuatan **Game Edukasi Pengenalan Budaya Maluku Menggunakan Teknologi Augmented Reality Berbasis Marker**, dari skema interaksi sistem hingga pustaka visual.

1 Arsitektur Sistem Web-AR

FRONTEND LAYER

A-Frame HTML, CSS 3D Engine, Tailwind, JavaScript untuk penanganan logika game dan pemetaan objek.

AR ENGINE (COMPUTER VISION)

AR.js (berbasis jsartoolkit) melacak koordinat fiducial marker matrix dari video input kamera.

ASSET & DATA MODEL

Hologram menggunakan elemen SVG terproteksi (untuk versi web ringan) atau file .gltf 3D untuk AR sesungguhnya.

Spesifikasi Marker 2D (Desain Pengenal)

Marker fisik harus memiliki kontras tinggi dengan bingkai hitam tebal dan pola asimetris di bagian dalam agar algoritma *Computer Vision* dapat mendeteksi sudut dan orientasi visual 3D secara presisi.

Pemberitahuan Hak Akses Kamera & Keamanan Sandbox

Aplikasi ini dijalankan dalam ruang aman (*sandboxed iframe*). Beberapa peramban memblokir integrasi perangkat keras kamera (`getUserMedia`) untuk alasan keamanan.

Kode Implementasi AR.js (Marker Hiro)

Untuk menjalankan Web-AR secara nyata di ponsel cerdas Anda, pastikan Anda memuat kode menggunakan modul A-Frame dan AR.js berikut: 

<script src="https://aframe.io/releases/1.2.0/aframe.min.js"></script>
<script src="https://raw.githack.com/AR-js-org/AR.js/master/aframe/build/aframe-ar.js"></script>

<a-scene embedded arjs="sourceType: webcam;">
  <!-- Pelacakan Marker Hiro bawaan AR.js -->
  <a-marker preset="hiro">
    <!-- Model 3D Budaya Maluku -->
    <a-entity gltf-model="baileo_model.gltf" scale="0.5 0.5 0.5"></a-entity>
  </a-marker>
  <a-entity camera></a-entity>
</a-scene>
